Topic: User Input

To receive input from the user, we use the Scanner class. This class is part of the java.util package and allows us to read different types of input such as strings, integers, and doubles.

Steps to use the Scanner class:

  1. Import the Scanner class: import java.util.Scanner;
  2. Create a Scanner object: Scanner sc = new Scanner(System.in);
  3. Use methods like nextLine(), nextInt(), etc., to read input.
    import java.util.Scanner;

    public class InputExample {
        public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter your name: ");
        String name = sc.nextLine();
        System.out.println("Hello, " + name + "!");
        }
    }

Output: If the user enters "John", the output will be:

    Enter your name: John
    Hello, John!

The Scanner class provides several methods to read user input. Here are some commonly used methods:

  • nextInt(): Reads an integer value from the user. If the input is not an integer, it throws an InputMismatchException.
  • next(): Reads a single word (up to the first whitespace) from the user input.
  • nextLine(): Reads an entire line of text, including spaces, until the user presses Enter.

Note: When using nextInt() or next() followed by nextLine(), you may encounter issues due to the newline character left in the input buffer. To avoid this, use an extra nextLine() to consume the leftover newline.

Topic: Basic Operators

Operators in Java are symbols that perform operations on variables and values. Java supports the following types of operators:

Operator Type Description Example
+ Arithmetic Addition 5 + 3 = 8
- Arithmetic Subtraction 5 - 3 = 2
* Arithmetic Multiplication 5 * 3 = 15
/ Arithmetic Division 6 / 3 = 2
% Arithmetic Modulus 5 % 3 = 2
== Relational Equal to 5 == 3 → false
!= Relational Not equal to 5 != 3 → true
> Relational Greater than 5 > 3 → true
< Relational Less than 5 < 3 → false
>= Relational Greater than or equal to 5 >= 3 → true
<= Relational Less than or equal to 5 <= 3 → false
&& Logical Logical AND (5 > 3) && (3 > 1) → true
|| Logical Logical OR (5 > 3) || (3 < 1) → true
! Logical Logical NOT !(5 > 3) → false
= Assignment Assign value x = 5
+= Assignment Add and assign x += 3 → x = x + 3
-= Assignment Subtract and assign x -= 3 → x = x - 3
*= Assignment Multiply and assign x *= 3 → x = x * 3
/= Assignment Divide and assign x /= 3 → x = x / 3

Example of arithmetic operations:

    public class OperatorsExample {
        public static void main(String[] args) {
        int a = 10, b = 5;
        System.out.println("Addition: " + (a + b));
        System.out.println("Multiplication: " + (a * b));
        System.out.println("Division: " + (a / b));
        }
    }

Output:

    Addition: 15
    Multiplication: 50
    Division: 2
Topic: Conditional Statements

Conditional statements allow you to execute a block of code based on a condition. The most common conditional statements in Java are:

  • if: Executes a block of code if the condition is true.
  • if-else: Executes one block of code if the condition is true, and another if it is false.

Example of an if-else statement:

    public class ConditionExample {
        public static void main(String[] args) {
        int age = 20;
        if (age >= 18) {
            System.out.println("You are an adult.");
        } else {
            System.out.println("You are a minor.");
        }
        }
    }

Output:

    You are an adult.
Topic: Loops

Loops are used to execute a block of code repeatedly as long as a condition is true. Java provides three types of loops:

  • for: Used when the number of iterations is known.

Example of a for loop:

    public class LoopExample {
        public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Iteration: " + i);
        }
        }
    }

Output:

    Iteration: 1
    Iteration: 2
    Iteration: 3
    Iteration: 4
    Iteration: 5
Topic: Output

In Java, the System.out.println() method is used to print text to the console. It automatically moves the cursor to the next line after printing.

Example:

    public class OutputExample {
        public static void main(String[] args) {
        System.out.print("Hello ");
        System.out.println("World!");
        System.out.printf("My name is %s and I am %d years old.", "John", 25);
        }
    }

Output:

    Hello World!
    My name is John and I am 25 years old.
